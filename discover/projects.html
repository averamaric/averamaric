Commission Anywhere, Control Any Robot 🥇

Siemens Connected Curriculum · Manchester Metropolitan University · 15/07/2026

Gold medal-winning project developed with a teammate for the Siemens Connected Curriculum Competition.

The project explored a new approach to virtual commissioning by enabling engineers to validate PLC and robot programs remotely, rather than relying solely on physical access to production equipment. This approach promotes global collaboration, reduces commissioning costs, minimizes travel-related emissions, and accelerates development cycles.

To achieve this, we developed a remote virtual commissioning environment that accurately mirrored the behavior of a real production line. The solution allowed engineers to use the Siemens Simatic Robot Programming Suite (SRPS) to test and validate robot programs in a realistic virtual environment, as if they were working directly on the shop floor.

We identified and acquired the relevant process signals from the output station of the Festo CP Lab mobile phone production line, implemented an OPC UA-based data acquisition system, transformed the data into MQTT messages for secure internet transmission, and streamed the information through a Node-RED pipeline into the Siemens virtual environment, enabling real-time synchronization between the physical and virtual systems.

Microgrid control testbed

Research Assistant · Manchester Metropolitan University · 12/2025 – 05/2026

Programmed an RTAC SEL 3555 PLC/gateway to implement PID control over a simulated microgrid running on OPAL-RT, with bidirectional DNP3 communication and Modbus TCP bidirectional communication, linking the microgrid simulation to a two-level industrial control testbed used for cybersecurity research.

Established the communications between the devices, programmed the RTAC as a PID control system, mapped and addressed signals inside the RTAC to keep data flowing cleanly between control levels and built the SCADA layer in Node-RED, with bidirectional comms back to the RTAC for real-time monitoring and control.
